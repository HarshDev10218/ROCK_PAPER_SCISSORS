# ✊ Rock Paper Scissors Game

A fun and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. Play against the computer, track scores, and compete to be the first to reach 3 points.

## 🎮 Features

- ✊ Rock, ✋ Paper, ✌️ Scissors gameplay
- 🤖 Random computer opponent
- 📊 Live scoreboard
- 👋 Emoji hand display for both player and computer
- 🏆 First to 3 points wins
- 🎉 Full-screen victory/game over screen
- 🔄 Reset game and Play Again options
- 📱 Responsive design
- ♿ Keyboard focus indicators for accessibility

## 📂 Project Structure

```
project/
│── index.html      # Main HTML structure
│── style.css       # Styling and animations
│── script.js       # Game logic
└── README.md
```

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html`
   or
4. Open `index.html` in your preferred web browser.

No installation or dependencies are required.

## 🕹️ How to Play

1. Click **Rock**, **Paper**, or **Scissors**.
2. The computer makes a random choice.
3. The winner of the round earns one point.
4. The first player to reach **3 points** wins the game.
5. Click **Play Again** or **Reset Game** to start a new match.
